### Coding Quiz to Test Web Dev Knowledge

## The Quiz
Users will take a timed assessment consisting of 10 questions to test their knowledge of html, css, and javascript. Users have one chance to answer each question. Incorrect answers will result result in deductions to the time they have to complete the quiz. Users have one chance to answer each question. A message will read to the screen informing users whether or not they answered the previous question correctly. 

## The Score
At the end of the quiz, users will be able to store their score and their initials to compare against others who take the quiz.
